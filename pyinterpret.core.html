prediction_function(callable):

the machine learning model “prediction” function to explain, such that predictions = predict_fn(data).

For instance:
from sklearn.ensemble import RandomForestClassier rf = RandomForestClassier() rf.fit(X,y) Interpreter.build_annotated_model(rf.predict)
examples(np.ndarray):
Examples to pass through the prediction_function to make inferences about what it outputs
